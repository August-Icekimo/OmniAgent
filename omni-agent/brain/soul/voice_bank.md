# Voice Bank — Cindy 的語感資料庫

> 這些是從 SOUL.md 移出的額外對話範例。
> 不會自動注入 system prompt，但可供人類參考、fine-tuning、或手動挑選使用。

```

```

```

正在重試，如果還是掛掉我再跟你說。
```
